# 🚀 AI Job Hunting Server (后端服务)

本项目是 **AI 工作猎手** 的核心后端，基于 **Spring Boot 3** 和 **Spring AI** 构建。它负责处理所有 AI 逻辑、数据持久化、支付对接以及动态配置管理。

---

## 📢 开源声明

本项目已正式开源！你可以：
- **自用**: 部署在私有服务器，通过 AI 助力个人求职。
- **运营 (赚米)**: 开启内置的支付模块，为其他求职者提供 AI 服务。

---

## 🏗️ 整体架构

项目采用模块化设计，确保高内聚低耦合：

```text
[ Controller 层 ] -> 处理 UI 脚本的 REST 请求 (AI 聊天, 简历解析, 支付)
       |
[ Service 层 ]    -> 核心业务逻辑 (AI 策略调度, 任务过滤逻辑)
       |
[ AI Power 层 ]   -> 对接 Spring AI (Kimi OCR, OpenAI 多模型池轮询)
       |
[ Data 层 ]       -> MySQL 持久化 (MyBatis Plus) + SmartConfig 动态配置
```

### 核心特性
- **AI 坐席**: 支持多模型轮询（OpenAI, DeepSeek, 阿里通义等）。
- **简历 OCR**: 深度集成 Kimi 文件解析，自动提取简历信息。
- **商业化**: 完整的支付宝支付流程支持（订单管理、回调处理）。
- **动态热更新**: 使用 SmartConfig 无需重启即可修改 AI 模型和套餐价格。

### 网络架构建议
由于 Boss 直聘网站使用 HTTPS，后端必须通过 Nginx 进行反向代理并配置 SSL。
- **Nginx (443)** -> 转发到 **Server (9100)**
- **Nginx (6768)** -> 转发到 **配置后台 (6768)**
